Avatars

MNAvatar‚Äã

The MNAvatar component is a customizable avatar display used in React Native applications. It can render either an image or fallback text within a circular or rectangular shape, and comes in multiple predefined sizes.

Features‚Äã

  • Supports image or text fallback.
  • Customizable size, shape, colors, and style.
  • Responsive design using size mappings.

Import‚Äã

import MNAvatar from '@/components/MNAvatar';

Props‚Äã

PropTypeDefaultDescription
size'xs' | 'sm' | 'md' | 'lg' | 'xl''md'Size of the avatar.
shape'circle' | 'rectangle''circle'Shape of the avatar.
imageUrlstringundefinedURL of the image to display.
fallbackTextstring''Text to show when image is not available.
backgroundColorstringPRIMARY_COLORBackground color of the avatar.
textColorstringWHITE_COLORText color for fallback text.
styleStyleProp<ViewStyle>undefinedCustom styles for the avatar container.

Usage‚Äã

Show an Avatar with Image‚Äã

<MNAvatar
    size="lg"
    shape="circle"
    imageUrl="https://example.com/avatar.jpg"
/>

Show an Avatar with Fallback Text‚Äã

<MNAvatar
    size="md"
    fallbackText="AB"
    backgroundColor="#4B5563"
    textColor="#F9FAFB"
/>

Rectangle Avatar‚Äã

<MNAvatar
    size="sm"
    shape="rectangle"
    fallbackText="CD"
/>

MNAvatarGroup Component‚Äã

The MNAvatarGroup component renders a group of avatars in a horizontal row with overlapping styles. It shows a limited number of visible avatars and displays a "+X" avatar if there are more than the maximum visible.

Import‚Äã

import MNAvatarGroup from '@/components/MNAvatarGroup';

Props‚Äã

PropTypeDefaultDescription
dataAvatarItem[]requiredArray of avatar data objects
maxVisiblenumber4Maximum number of visible avatars
size'xs' | 'sm' | 'md' | 'lg' | 'xl''md'Size of each avatar
shape'circle' | 'rectangle''circle'Shape of each avatar
overlapnumber16Number of pixels each avatar overlaps the previous
styleStyleProp<ViewStyle>undefinedCustom style for the avatar group container

Usage‚Äã

Basic Avatar Group‚Äã

const avatars = [
    {id: 1, imageUrl: 'https://example.com/avatar1.jpg'},
    {id: 2, fallbackText: 'AB', backgroundColor: '#ff5722', textColor: '#fff'},
    {id: 3, imageUrl: 'https://example.com/avatar3.jpg'},
    {id: 4, fallbackText: 'CD'},
    {id: 5, fallbackText: 'EF'},
];

<MNAvatarGroup
    data={avatars}
    maxVisible={4}
    size="sm"
    shape="circle"
    overlap={12}
/>

üîç Behavior‚Äã

  • Displays up to maxVisible avatars from the data array.
  • Avatars overlap horizontally by the overlap amount.
  • If there are more avatars than maxVisible, the last avatar shows a +N count of remaining avatars.
  • Supports both image and fallback text avatars.
  • Shape and size customizable for all avatars.

Types‚Äã

AvatarItem‚Äã

type AvatarItem = {
    id: string | number;
    imageUrl?: string;
    fallbackText?: string;
    backgroundColor?: string;
    textColor?: string;
};

MNAvatarGroupProps‚Äã

type MNAvatarGroupProps = {
    data: AvatarItem[];
    maxVisible?: number;
    size?: 'xs' | 'sm' | 'md' | 'lg' | 'xl';
    shape?: 'circle' | 'rectangle';
    overlap?: number;
    style?: StyleProp<ViewStyle>;
};

Styling‚Äã

const styles = StyleSheet.create({
    container: {
        flexDirection: 'row',
    },
    avatarWrapper: {
        zIndex: 1,
    },
});
